
Image-View
=====

Image View is a simple Django app to show image in admin

Detailed documentation is in the "docs" directory.



Quick start
-----------

1. Add "django-image-view" to your INSTALLED_APPS setting like this::

    INSTALLED_APPS = [
        ...
        'django-image-view',
    ]
